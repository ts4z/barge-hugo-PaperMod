# barge-hugo-PaperMod

The [BARGE web site](https://www.barge.org/) is a Hugo site
with its [Hugo](https://gohugo.io/) source code at [barge.org](github.com/ts4z/barge.org).
It was built with the [PaperMod](https://github.com/adityatelange/hugo-PaperMod/) theme.
This is our fork for our local tweaks and tracking.
